# Event-Calendar-System-for-Colombo-Area
Designed a web-based centralized platform to help users easily explore and engage with events through clear scheduling, location details, and real-time availability. The project focuses on improving event accessibility, supporting organizers and artists, and enhancing user engagement through seamless booking and feedback features.

1. To provide a centralized platform for discovering and exploring events happening in Colombo.

2. To help users easily view event details such as date, time, location, and ticket availability.

3. To offer an interactive event calendar highlighting holidays, festivals, and public celebrations.

4. To enable online ticket booking with real-time availability updates and QR-based confirmation.

5. To allow users to manage bookings, including viewing and canceling reservations.

6. To support artists by showcasing profiles and promoting upcoming performances.

7. To allow users to share reviews and ratings to guide event participation decisions.

8. To assist users in locating event venues using an integrated map-based location search.

9. To enable event organizers to add, update, and remove events efficiently.

10. To provide administrators with full system control through a secure management dashboard.

Technologies: Eclipse IDE for Enterprise Java and Web Developers, MySQL, Apache Tomcat, Jakarta Servlet and
JSP APIs, JavaMail API, OpenPDF, Leaflet.js, FullCalendar.js, TailwindCSS
